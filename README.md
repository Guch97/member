# 会员管理系统

>##### 技术实现:Easy-mock+axios+vue-cli+element-ui
>##### 实现效果:登录退出,权限拦截,会员管理,员工管理等 新增查询删除等功能

## 本地预览

**下载**

git clone https://github.com/gchgch789/member.git

**运行**

npm install

npm run serve
